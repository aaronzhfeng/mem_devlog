# Decision Memos

Major methodological decisions. Each file has YAML frontmatter with `id` (D-xxx), `dag_nodes`, and typed `links`.

Template:

```yaml
---
id: D-001
title: "Decision title"
date: YYYY-MM-DD
dag_nodes: [I01, I02]
links:
  evidence_for: []
  evidence_against: []
  derived_from: []
  related_to: []
---
```
